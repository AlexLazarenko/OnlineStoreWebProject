# OnlineStoreWebProject
Online store. Web MVC application with layered architecture.
 * **Features**
      - Using database PostgreSQL
      - Servlet
      - Custom connection pool and proxy connections
      - JSP
      - Maven
      - JSTL
      - Log4j2
      - Double validation
      - XSS attacks protection
      - Sending email via SMTP
      - Localization: EN, RU
